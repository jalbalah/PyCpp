
PyCpp converts template code that is modeled 
after python syntax into C++ code by mapping directly.

lessons:

`./lessons/lesson1`

`./lessons/lesson2`

class properties:  
`./lessons/lesson3`

shared (static) class variables:  
`./lessons/lesson4`

lists and loops:  
`./lessons/lesson5`
